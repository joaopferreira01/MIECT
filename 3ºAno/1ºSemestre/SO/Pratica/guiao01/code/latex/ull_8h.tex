\doxysubsection{Detailed Description}
A unidirectional linked-\/list, whose elements are registers composed of a 32-\/bit unsigned integer, representing a student number, and a string, used to store the student name. 

The following operations are defined\+: \begin{DoxyItemize}
\item insertion of a register \item query of a name by number \item removal of a register \item reset of the list \item print the list contents sorted by number or name \item load list from file\end{DoxyItemize}
