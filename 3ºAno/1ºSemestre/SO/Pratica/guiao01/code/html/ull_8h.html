Detailed Description

A unidirectional linked-list, whose elements are registers composed of a 32-bit unsigned integer, representing a student number, and a string, used to store the student name.

The following operations are defined:

  • insertion of a register
  • query of a name by number
  • removal of a register
  • reset of the list
  • print the list contents sorted by number or name
  • load list from file
